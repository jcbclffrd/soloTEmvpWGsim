## 🚀 Quick Fresh Run Test

Complete test from scratch (allows overnight run):

```bash
# Day 1: Setup
cd /your/workspace
git clone https://github.com/jcbclffrd/soloTEmvpWGsim.git
cd soloTEmvpWGsim

# Configure account
bash configure_hpc_account.sh  # Enter your lab name

# Setup conda (interactive)
srun -c 4 -p free --pty /bin/bash -i
module load miniconda3/25.11.1
. ~/.mycondainit
conda env create -f environment.yml
exit

# Submit everything with dependencies
cd setup
REF_JOB=$(sbatch sbatch_00_setup_references.sh | awk '{print $4}')
STAR_JOB=$(sbatch --dependency=afterok:$REF_JOB sbatch_01_build_star_index.sh | awk '{print $4}')
SOLO_JOB=$(sbatch --dependency=afterok:$STAR_JOB sbatch_02_install_solote.sh | awk '{print $4}')
cd ..
PIPE_JOB=$(sbatch --dependency=afterok:$SOLO_JOB sbatch_run_pipeline.sh | awk '{print $4}')

echo "Jobs submitted:"
echo "  References: $REF_JOB"
echo "  STAR Index: $STAR_JOB"
echo "  soloTE:     $SOLO_JOB"
echo "  Pipeline:   $PIPE_JOB"

# Day 2: Check results
cat validation_report/validation_metrics.tsv
cat PIPELINE_STATUS.md
```

**Timeline**:
- References: ~20 min
- STAR index: ~40 min  
- soloTE: ~5 min
- Pipeline: ~6 min
- **Total**: ~71 min (1.2 hours)
